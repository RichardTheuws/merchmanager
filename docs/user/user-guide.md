## üéµ Band Management

### Creating Bands
1. Navigate to **MerchManager ‚Üí Bands**
2. Click **Add New Band**
3. Fill in required information:
   - **Band Name**: Official name of the band
   - **Slug**: URL-friendly identifier
   - **Contact Email**: Primary contact email
   - **Contact Phone**: Primary phone number
   - **Description**: Band bio and information

### Band Details
- **Members**: Band members and roles
- **Genres**: Music genres and styles
- **Social Media**: Links to social profiles
- **Website**: Official band website
- **Payment Details**: Bank/Payment information

### Managing Multiple Bands
- Use the band switcher in the admin bar
- Filter reports and sales by band
- Set user permissions per band
- Export band-specific data

### Adding Products
1. Go to **MerchManager ‚Üí Merchandise**
2. Click **Add New Item**
3. Complete product details:
   - **Product Name**: Clear, descriptive name
   - **Description**: Product details and features
   - **Price**: Selling price (excluding tax)
   - **Cost Price**: Wholesale/manufacturing cost
   - **Initial Stock**: Starting inventory quantity
   - **Low Stock Threshold**: Alert level
   - **Band**: Associated band
   - **Categories**: Product categories

### Product Variations
- **Sizes**: S, M, L, XL, etc.
- **Colors**: Various color options
- **Materials**: Different material types
- **Designs**: Multiple design variants

### Product Images

- Optimize images for web

## üí∞ Sales Management

### Recording Sales
1. Navigate to **MerchManager ‚Üí Record Sale**
2. Select the sales context:
   - **Live Show**: Concert or event
   - **Online Store**: Web sales
   - **Retail**: Physical store
   - **Other**: Custom sales type

3. Add items to sale:
   - Select merchandise items
   - Enter quantities
   - Apply discounts if needed
   - Review totals

4. Complete sale:
   - Select payment method
   - Add customer notes
   - Record sale

### Sale Details
- **Date/Time**: Sale timestamp
- **Location**: Show venue or location
- **Sales Person**: Who made the sale
- **Payment Type**: Cash, card, transfer, etc.
- **Tax Calculation**: Automatic tax handling
- **Discounts**: Percentage or fixed amount

### Editing Sales
- View sales history
- Edit existing sales (with permissions)
- Add notes to sales
- Process refunds
