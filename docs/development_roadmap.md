# Merchandise Sales Plugin - Development Roadmap

## 1. Overview

This document outlines the development roadmap for the Merchandise Sales Plugin, providing a timeline and prioritization for feature implementation and enhancements.

## 2. Current Version (1.0.1)

The current version includes the following implemented features:

- Custom post types (Band, Tour, Show, Merchandise, Sale, Sales Page)
- User roles and permissions
- Band management
- Tour management
- Show management
- Merchandise management
- Sales management
- Basic reporting
- Frontend management
- CSV import/export functionality
- Stock management
- Dummy content generation
- Edit functionality
- Sales page generation
