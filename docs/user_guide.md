# Merchandise Sales Plugin - User Guide

## 1. Introduction

Welcome to the Merchandise Sales Plugin User Guide. This guide will help you understand how to use the plugin to manage your band's merchandise sales during tours and events.

### 1.1 About the Plugin

The Merchandise Sales Plugin is a WordPress plugin designed specifically for bands and music artists to manage their merchandise sales. It provides a comprehensive solution for tracking inventory, recording sales, managing tours, and generating reports.

### 1.2 Key Features

- Multi-band support
- Tour and show management
- Merchandise inventory management
- Sales recording and tracking
- Dynamic sales page generation for events
- Detailed reporting
- CSV import/export functionality
- Stock level monitoring and alerts

### 1.3 User Roles

The plugin implements three user roles:

- **MSP Management**: Full access to all plugin functions
- **MSP Tour Management**: Can manage tours and merchandise
- **MSP Merch Sales**: Can register sales

## 2. Getting Started

### 2.1 Installation

1. Download the plugin ZIP file
2. Log in to your WordPress admin panel
3. Go to Plugins > Add New
4. Click "Upload Plugin"
5. Choose the ZIP file and click "Install Now"
6. After installation, click "Activate Plugin"

### 2.2 Initial Setup

After activating the plugin, you'll need to complete the initial setup:

1. Go to Merchandise Sales > Settings
2. Configure general settings (currency, date format, etc.)
3. Set up user roles and permissions
4. Create your first band

### 2.3 Dashboard Overview

The plugin adds a new menu item "Merchandise Sales" to your WordPress admin menu. This menu provides access to all plugin features:

- Dashboard: Overview of sales, inventory, and upcoming shows
- Bands: Manage bands
- Tours: Manage tours and shows
- Merchandise: Manage merchandise items
- Sales: Record and view sales
- Sales Pages: Manage generated sales pages
- Reports: View and export reports
- Settings: Configure plugin settings

## 3. Band Management

### 3.1 Creating a Band

1. Go to Merchandise Sales > Bands
2. Click "Add New Band"
3. Enter the band name and description
4. Add contact information (name, email, phone)
5. Upload a band logo (optional)
6. Add social media links (optional)
7. Click "Save Band"

### 3.2 Editing a Band

1. Go to Merchandise Sales > Bands
2. Find the band you want to edit
3. Click "Edit" or click on the band name
4. Make your changes
5. Click "Update Band"

### 3.3 Deleting a Band

1. Go to Merchandise Sales > Bands
2. Find the band you want to delete
3. Click "Delete"
4. Confirm the deletion

**Note**: Deleting a band will also delete all associated tours, shows, merchandise, and sales data. This action cannot be undone.

### 3.4 Assigning Users to a Band

1. Go to Merchandise Sales > Bands
2. Find the band you want to assign users to
3. Click "Manage Users"
4. Select users from the dropdown
5. Assign roles to each user
6. Click "Save Assignments"

## 4. Tour Management

### 4.1 Creating a Tour

1. Go to Merchandise Sales > Tours
2. Click "Add New Tour"
3. Select a band
4. Enter the tour name and description
5. Set the start and end dates
6. Add any additional notes
7. Click "Save Tour"

### 4.2 Adding Shows to a Tour

1. Go to Merchandise Sales > Tours
2. Find the tour you want to add shows to
3. Click "Manage Shows"
4. Click "Add New Show"
5. Enter the show details:
   - Show name
   - Date and time
   - Venue name
   - Venue address
   - Contact information
6. Click "Save Show"

### 4.3 Importing Tour Schedule

1. Go to Merchandise Sales > Tours
2. Click "Import Tour Schedule"
3. Select a band
4. Download the CSV template
5. Fill in the template with your tour schedule
6. Upload the completed CSV file
7. Map the CSV columns to the plugin fields
8. Click "Import"

### 4.4 Exporting Tour Schedule

1. Go to Merchandise Sales > Tours
2. Find the tour you want to export
3. Click "Export"
4. Choose the export format (CSV or Excel)
5. Click "Download"

## 5. Merchandise Management

### 5.1 Adding Merchandise Items

1. Go to Merchandise Sales > Merchandise
2. Click "Add New Item"
3. Select a band
4. Enter the item details:
   - Item name
   - Description
   - SKU (optional)
   - Price
   - Size (if applicable)
   - Color (if applicable)
   - Initial stock level
   - Low stock threshold
   - Supplier information (optional)
   - Cost per unit (optional)
   - Category (optional)
5. Upload an image (optional)
6. Click "Save Item"

### 5.2 Managing Stock Levels

1. Go to Merchandise Sales > Merchandise
2. Find the item you want to update
3. Click "Update Stock"
4. Enter the new stock level or the quantity to add/subtract
5. Add a note explaining the stock change (optional)
6. Click "Update"

### 5.3 Importing Merchandise Data

1. Go to Merchandise Sales > Merchandise
2. Click "Import Merchandise"
3. Select a band
4. Download the CSV template
5. Fill in the template with your merchandise data
6. Upload the completed CSV file
7. Map the CSV columns to the plugin fields
8. Click "Import"

### 5.4 Exporting Merchandise Data

1. Go to Merchandise Sales > Merchandise
2. Click "Export"
3. Select a band
4. Choose the export format (CSV or Excel)
5. Click "Download"

## 6. Sales Management

### 6.1 Recording Sales

1. Go to Merchandise Sales > Sales
2. Click "Record Sale"
3. Select a band
4. Select a show or sales page (optional)
5. Add items to the sale:
   - Select merchandise item
   - Enter quantity
   - The price will be automatically filled (can be overridden)
6. Select payment type (cash, card, etc.)
7. Add any notes (optional)
8. Click "Save Sale"

### 6.2 Viewing Sales History

1. Go to Merchandise Sales > Sales
2. Use the filters to narrow down the results:
   - Band
   - Date range
   - Show
   - Sales page
   - Payment type
3. Click "Apply Filters"
4. View the sales data in the table

### 6.3 Editing or Deleting Sales

1. Go to Merchandise Sales > Sales
2. Find the sale you want to edit or delete
3. Click "Edit" to modify the sale or "Delete" to remove it
4. If editing, make your changes and click "Update Sale"
5. If deleting, confirm the deletion

**Note**: Editing or deleting a sale will automatically adjust the stock levels accordingly.

## 7. Sales Pages

### 7.1 Generating a Sales Page

1. Go to Merchandise Sales > Sales Pages
2. Click "Generate New Page"
3. Select a band
4. Select a show
5. Enter a title for the sales page
6. Select merchandise items to include
7. Set an access code (optional)
8. Set an expiry date (optional)
9. Click "Generate Page"

### 7.2 Using a Sales Page

1. Go to Merchandise Sales > Sales Pages
2. Find the sales page you want to use
3. Click "View" to see the page
4. Share the URL with your sales staff
5. If an access code is set, provide it to your staff

The sales page provides a simple interface for recording sales during an event:

1. Enter the access code (if required)
2. Select merchandise items and quantities
3. Select payment type
4. Click "Record Sale"

### 7.3 Managing Sales Pages

1. Go to Merchandise Sales > Sales Pages
2. Find the sales page you want to manage
3. Available actions:
   - View: Open the sales page
   - Edit: Modify the sales page settings
   - Deactivate/Activate: Toggle the page status
   - Delete: Remove the sales page

## 8. Reporting

### 8.1 Sales Reports

1. Go to Merchandise Sales > Reports
2. Select "Sales Report"
3. Set the report parameters:
   - Band
   - Date range
   - Tour
   - Show
   - Group by (day, week, month, merchandise)
4. Click "Generate Report"
5. View the report data and charts
6. Export the report (PDF, CSV, Excel) if needed

### 8.2 Inventory Reports

1. Go to Merchandise Sales > Reports
2. Select "Inventory Report"
3. Set the report parameters:
   - Band
   - Show low stock items only (optional)
   - Categories (optional)
4. Click "Generate Report"
5. View the report data
6. Export the report (PDF, CSV, Excel) if needed

### 8.3 Tour Reports

1. Go to Merchandise Sales > Reports
2. Select "Tour Report"
3. Select a band
4. Select a tour
5. Click "Generate Report"
6. View the report data and charts
7. Export the report (PDF, CSV, Excel) if needed

## 9. Settings

### 9.1 General Settings

1. Go to Merchandise Sales > Settings
2. Select the "General" tab
3. Configure the following settings:
   - Currency
   - Date format
   - Time format
   - Default low stock threshold
   - Default sales page expiry period

### 9.2 User Roles and Permissions

1. Go to Merchandise Sales > Settings
2. Select the "Users" tab
3. Configure permissions for each user role
4. Click "Save Changes"

### 9.3 Email Notifications

1. Go to Merchandise Sales > Settings
2. Select the "Notifications" tab
3. Configure email notifications for:
   - Low stock alerts
   - Sales summaries
   - New sales page generation
4. Click "Save Changes"

### 9.4 Advanced Settings

1. Go to Merchandise Sales > Settings
2. Select the "Advanced" tab
3. Configure advanced settings:
   - Data retention period
   - Debug mode
   - CSV import/export settings
4. Click "Save Changes"

## 10. Mobile Usage

The plugin is designed to be responsive and work well on mobile devices, making it ideal for recording sales at events.

### 10.1 Mobile Sales Recording

1. Access your WordPress site on a mobile device
2. Log in with your credentials
3. Go to Merchandise Sales > Sales
4. Click "Record Sale"
5. Follow the same process as on desktop

### 10.2 Using Sales Pages on Mobile

1. Access the sales page URL on a mobile device
2. Enter the access code (if required)
3. Record sales using the mobile-optimized interface

## 11. Troubleshooting

### 11.1 Common Issues

#### Sales Not Recording
- Check that you have sufficient permissions
- Verify that the merchandise item is active
- Check that the stock level is not zero

#### Stock Levels Not Updating
- Check that the merchandise item exists
- Verify that the sale was recorded successfully
- Check for any error messages in the logs

#### Reports Not Generating
- Verify that you have sales data for the selected period
- Check that you have sufficient permissions
- Try clearing your browser cache

### 11.2 Getting Help

If you encounter issues not covered in this guide:

1. Check the plugin documentation
2. Visit the support forum
3. Contact the plugin developer

## 12. Best Practices

### 12.1 Data Management

- Regularly backup your WordPress database
- Export important data periodically
- Archive old tours and shows to keep the interface clean

### 12.2 Security

- Use strong passwords for all user accounts
- Regularly update WordPress and all plugins
- Limit access to the plugin based on user roles
- Use access codes for sales pages

### 12.3 Performance

- Keep your merchandise catalog organized
- Archive old sales data periodically
- Use the reporting features to identify trends and optimize inventory

## 13. Glossary

- **Band**: A music artist or group
- **Tour**: A series of shows or events
- **Show**: A single performance at a specific venue and date
- **Merchandise**: Items for sale (t-shirts, CDs, etc.)
- **Sales Page**: A temporary page for recording sales at a specific event
- **Stock Level**: The quantity of a merchandise item available for sale
- **Low Stock Threshold**: The quantity at which a low stock alert is triggered
- **SKU**: Stock Keeping Unit, a unique identifier for merchandise items
