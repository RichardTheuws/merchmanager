# Plugin Assets

These assets are used for the WordPress.org plugin directory.

## WordPress.org SVN

When publishing to WordPress.org, copy these files to the SVN `/assets` directory (top-level, next to `/trunk`):

- `icon.svg` – SVG plugin icon (merchandise + bands theme)
- `icon-128x128.png` – PNG fallback at 128×128
- `icon-256x256.png` – Retina PNG fallback at 256×256

Optional:

- `banner-772x250.png` – Plugin banner (772×250 px)
- `banner-1544x500.png` – Retina banner (1544×500 px)
- `screenshot-1.png`, `screenshot-2.png`, etc. – Screenshots with captions in readme.txt
